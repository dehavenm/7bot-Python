# 7bot-Python

A Python library for controlling the 7Bot robot arm from a host PC.

The 7Bot is a desktop six-axis robot arm released on Kickstarter in 2015. It is driven by specialized digital servos connected to an Arduino Due in the base of the robot. The robot can be controlled from a PC by sending and receiving data through the USB link in the Arduino board.

Arduino software (Can be downloadwed at https://www.arduino.cc/en/Main/Software)
